1. Getting Started

  1. Install JxL Scheduler on your iPhone.
  2. Open the app. A local profile is created automatically.
  3. Use Schedule, Routes, Messages, Uploads, and Profile as needed.
  4. Optional iCloud sharing can be enabled when you want to collaborate with trusted members.

2. Availability and Schedules

  • Use the plus button in Schedule to add availability.
  • Tap an existing availability card to edit it.
  • Swipe left on an availability card to delete it.
  • Recurring availability can use start and end dates.

3. Routes Help

  • Create a route from the Routes tab using the plus button.
  • Add stops by searching for an address or selecting a map location.
  • Reorder stops from the route detail or edit view.
  • Use Apple Maps or Google Maps links to open route directions when available.

4. Messages Help

  • Messages can be used as notes to yourself or for group communication when sharing is enabled.
  • Tap outside the composer to dismiss the keyboard.
  • Use delete or clear controls to remove messages you created.

5. Uploads Help

  • Uploads are stored for use inside the app.
  • Shared upload features depend on the current app version and iCloud availability.
  • If a file is missing, confirm the file is still available on the device or in the shared record.

6. iCloud Sharing Help

If shared group data does not appear on another device:

  • Make sure both devices are signed in to iCloud.
  • Open JxL Scheduler on both devices.
  • Confirm the receiving member accepted the iCloud share invitation.
  • Try foregrounding the app again to trigger sync.

7. Frequently Asked Questions

Do I need a JxL account?

No. JxL Scheduler does not require a separate app sign-in.

Can I use the app offline?

Yes. The app is local-first, so schedules, routes, messages, and uploads can be used locally. Shared sync requires iCloud and network availability.

Can I use the app without location permission?

Yes. Location is optional unless you want current-location route features.

How do I clear old local test data?

Use reset options in the app when available, or delete and reinstall the app on the device.

8. Contact Support

Email support@edentv.us with your device model, iOS version, app version, and a short description of the issue.

Support requests for JxL Scheduler are typically answered within 2 business days.
