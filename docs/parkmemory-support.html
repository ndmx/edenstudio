1. Getting Started

  1. Install Park Memory Hub on your iPhone.
  2. Open the app and set your display name in Profile.
  3. Create a new circle or join an existing circle from an invite link or circle code.
  4. Use Memories, Radar, Planner, and Profile as needed.

2. Permissions

  • Photos: Needed to attach photos to memories.
  • Location: Needed only if you want current-location tagging or radar sharing.
  • iCloud: Needed for shared memories, planner updates, and circle sync through CloudKit.

3. iCloud Sync Help

If updates seem slow or another device does not show the latest shared content:

  • Make sure both devices are signed in to iCloud.
  • Open Park Memory Hub on both devices.
  • Use the manual Sync with iCloud button in Profile or the feature toolbar.
  • Confirm both devices are using the same circle code.

4. Memories Help

  • If a memory is missing on another device, confirm it was created in the same circle.
  • If a photo cannot be attached, confirm the app still has Photos permission.
  • If a memory has multiple photos, the receiving device may need a moment for the shared media to appear.

5. Planner Help

  • Circle shares with everyone in the circle.
  • Some Members shares only with the selected people and the creator.
  • Only Me keeps the plan visible only on the creator's device.

6. Radar Help

  • Each person must enable location sharing on their own device.
  • Tapping a member opens Apple Maps with their shared coordinates when available.
  • If radar is empty, verify the device has Location permission and that the member is sharing.

7. Frequently Asked Questions

Do both phones need the same Apple ID?

No. Separate Apple IDs are supported. In fact, separate Apple IDs are better for real-world group testing.

Can I leave a circle?

Yes. In Profile, use Leave Circle. This device will move into a new personal circle with a new circle code.

Can I use the app without location services?

Yes. Location is optional unless you want current-location tagging or radar sharing.

How do I clear old local test data on a device?

Delete Park Memory Hub from that device and reinstall it. That clears the device's local circle, memories, plans, profile data, and sync cache before you set it up again.
